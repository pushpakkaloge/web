3) Hardware/Software Tools, Kits and Equipments


  • E-Yantra lab setup
  • Blackfinn processor based DIP lab (Real time video processing set up)
  • PCB design and development set up.
  • High end experimentation kits like UHF Antenna kits, DSP boards, ARM 9 board, OMAP138 board, ZYNQ SoC board, Spartan6, Vertex 5 and 7 based FPGA kits with all peripherals.
  • Advanced software tools like Mentor graphics HEP1, HEP2, Xilinx ISE -14.2, MATLAB and Simulink, MULTISIM and ULTIBOARD etc.
  • Multidisciplinary set up like automation studio
  • Advanced equipments like OTDR, Spectrum analyzer, DSO, logic analyzer
  • Laboratories with high speed LAN network facilities with internet.
